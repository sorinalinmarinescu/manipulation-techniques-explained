# Case Files Index & Technique Cross-Reference

A study aid for the 10 **Case Files** (Episodes 63–72) — the long, two-part worked examples where the techniques from across the series combine in the wild.

Use this two ways:
- **Forward** (Table 1): pick a campaign, see every technique it uses (with episode numbers).
- **Reverse** (Table 2): pick a technique, see which campaigns demonstrate it.

Every Case File is a **composite dramatization** (Part 1: the campaign unfolds · Part 2: the breakdown), framed for recognition and self-defense. The political ones are deliberately non-partisan.

## How to use this in production

- **Teaching a technique?** Look it up in Table 2 and pull the matching Case File for a concrete, dramatized example.
- **Producing a Case File?** Use Table 1 as a shot-list of the "Part 2: breakdown" beats.
- **Building a "best of" or trailer?** The most technique-dense Case Files are **63** (cult), **67** (disinformation), and **69** (miracle cure) — each stacks a dozen-plus techniques.
- **The recurring spine across all Case Files:** *isolation* (cutting you off from outside anchors), *urgency / sunk cost* (rushing you and trapping your investment), and *manufactured trust* (love bombing, plain folks, borrowed authority, fake crowds). If you teach nothing else, teach those three — they appear almost everywhere.
